# Sos Bichos Mobile
### Application developed to ONG Sos Bichos in the city Ponta Grossa.
- Application for donations and adoption of animals

### Technologies
- React Native.
- React Hooks.
- TypeScript.
- Styled components.
- Redux.
- Redux Sagas.

### Getting started
 
 - After initial installation of react native (https://reactnative.dev/docs/getting-started):
 
 1 - yarn install </br>
 2 - yarn android </br>
 3 - yarn start </br>
